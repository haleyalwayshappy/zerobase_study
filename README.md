#### scss 
`sudo npm i -g sass` 
- sass 설치 
- vscode에서 live sass compiler 설치 - 확장설정 - setting.json 
- setting.json 에 해당코드 넣어주기
 ``` 
   "liveSassCompile.settings.formats": [
        {
            "format": "expanded",
            "extensionName": ".css",
            "savePath": "./assets/css"
        }
    ]
 ``` 
이 코드는 sass,scss를 css파일로 컴파일 해줄때 어느 경로에 저장할 것인지 지정해주는 것 
`"savePath": "./assets/css"` 이 부분 중요!
